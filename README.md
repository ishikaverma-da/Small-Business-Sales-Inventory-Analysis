# Small-Business-Sales-Inventory-Analysis
SQL &amp; Power BI Project analyzing sales, inventory, customer ,and product data

## Project Description
This project demonstrates a complete analysis of small business sales data using SQL and Power BI.  
It includes:
- SQL queries for data extraction and analysis
- CSV outputs for each query
- Power BI dashboards for visual insights
- A PDF presentation summarizing the project

---

## SQL Queries and Results
All 14 SQL queries and their corresponding results are available in the `sql` folder:
- Q1_query.sql → Q1_result.csv
- Q2_query.sql → Q2_result.csv
- …
- Q14_query.sql → Q14_result.csv

Each query extracts meaningful insights from the dataset and is documented in the respective `.sql` and `.csv` files.

---

## Data
All raw datasets are stored in the `data` folder:
- customer.csv
- products.csv
- order.csv
- orderdetails.csv
