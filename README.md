# Small Business Sales Analysis

## Project Overview
This project analyzes small business sales data to identify trends, customer behavior, and product performance.

## Tools Used
- Excel
- SQL
- Power BI

## Dataset
The dataset includes customer, products, order, and orderdetails stored in CSV format.

## Key Analysis
- Sales trends over time
- Top-performing products
- Customer purchase behavior

Project in progress. SQL analysis and Power BI dashboard will be added.
